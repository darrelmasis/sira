# SIRA

**Sistema de Información para Registro Avícola**

SIRA es una aplicación para registrar la mortalidad de aves en granjas. Está diseñada para trabajar tanto con conexión a internet como sin ella, pensada para el uso diario en campo.

---

## ¿Qué puedo hacer con SIRA?

- **Registrar mortalidad** — Ingresa la mortalidad del día seleccionando granja, galpón y lote. Los datos se guardan automáticamente aunque no tengas internet.
- **Ver mi historial** — Consulta todos los registros que has enviado y su estado de sincronización.
- **Administrar catálogos** — Si tienes permisos, puedes crear y editar granjas, galpones, lotes y alojamientos.
- **Gestionar usuarios** — Los administradores pueden crear usuarios, asignar roles y definir qué permisos tiene cada rol.
- **Personalizar mi perfil** — Cambia tu contraseña, elige un avatar y revisa tus datos personales.
- **Sincronizar datos** — La app sincroniza automáticamente los registros cuando hay conexión. También puedes forzar la sincronización manualmente.

---

## Roles de usuario

| Rol | ¿Qué puede hacer? |
|---|---|
| **Desarrollador** | Acceso completo a todo |
| **Administrador** | Administra catálogos, usuarios, roles y ve todos los registros |
| **Supervisor** | Registra mortalidad, ve catálogos y puede sincronizar manualmente |
| **Operador** | Registra mortalidad del día actual y consulta catálogos |

---

## Comenzar

1. Abre la aplicación en tu navegador o en tu dispositivo móvil.
2. Inicia sesión con tu usuario y contraseña.
3. Usa el menú lateral para navegar entre las secciones.
4. Para registrar mortalidad, ve a la sección **Mortalidad**, completa el formulario y guarda.
5. Revisa el estado de tus registros en **Historial**.

> Puedes instalar SIRA como una aplicación en tu teléfono o computadora desde el mismo navegador.

## Preguntas frecuentes

**¿Qué pasa si no tengo internet cuando registro?**
Los datos se guardan en el dispositivo y se enviarán automáticamente cuando haya conexión.

**¿Cuánto tarda en sincronizarse?**
La app intenta sincronizar automáticamente cada 30 segundos cuando estás en línea.

**¿Puedo editar un registro ya enviado?**
Actualmente los registros no se pueden editar una vez sincronizados.

**¿Cómo sé si mis datos se sincronizaron correctamente?**
En el Historial puedes ver el estado de cada registro.
